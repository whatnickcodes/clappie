## Common Benchmarks on backtes.to

The benchmark dropdown uses the **same ETF/bond/stock catalog** as the main portfolio dropdown.
There are no bare "named indices" — to benchmark against an index, select an ETF that tracks it.

> **Search rule**: Use ISIN for benchmark ETFs, same as portfolio ETFs. The benchmark dropdown
> accepts the same search format: `[Name] | [ISIN] | Da [MM/YYYY]`.
