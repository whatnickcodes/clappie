[SHARED RULES]
PREMISE:
Chain topology demo. Five agents in a line: A → B → C → D → E.
Each agent can ONLY talk to their immediate neighbor. A message must relay hop-by-hop from A to E.
The goal: A starts with a secret phrase. It must reach E intact through 4 hops.

MECHANICS:
- hops: Number of relays completed (0-4).
- message: The current version of the relayed message.
- status: Each agent's state (waiting, relaying, done).

RULES:
- A broadcasts the starting phrase, then DMs B.
- Each agent receives from their left neighbor, may add brief context, then DMs their right neighbor.
- Agents CANNOT skip hops. B cannot talk to D. C cannot talk to A. Only adjacent neighbors.
- When E receives the message, they broadcast the final version.
- Compare start vs end. Telephone game — see how much drifted.
- One message per hop. Keep it to 1-2 sentences. Fast relay.
