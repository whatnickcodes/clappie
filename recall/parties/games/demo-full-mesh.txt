[SHARED RULES]
PREMISE:
Full mesh topology demo. Five agents, all interconnected. Everyone can talk to everyone.
The goal: reach consensus on a question through open discussion. No leader, no hierarchy.
Pure democratic deliberation. Majority wins.

MECHANICS:
- edges: Total connections (10 — every pair connected).
- consensus: Percentage of agents that agree.
- proposal: The current leading proposal.
- status: Each agent's state (discussing, voted, done).
- position: Each agent's current stance.

RULES:
- Any agent can propose an answer. Broadcast it to all.
- Any agent can counter-propose, argue, or agree. DM anyone directly.
- No designated leader. No turn order. It's a free-for-all.
- When an agent is ready, they set their position to their final answer.
- Once all 5 have set positions, tally consensus.
- If 3+ agree on the same answer, that's the consensus. Broadcast it.
- If no majority after 5 exchange rounds, the most popular answer wins by plurality.
- Keep messages to 1-2 sentences. Fast deliberation.
