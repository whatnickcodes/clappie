## Step 2: Fetch outside weather from Open-Meteo

```
curl -s "https://api.open-meteo.com/v1/forecast?latitude=44.7824&longitude=10.3373&current=temperature_2m,relative_humidity_2m,apparent_temperature,wind_speed_10m,weather_code,cloud_cover,surface_pressure,dew_point_2m,uv_index,shortwave_radiation&hourly=sunshine_duration&timezone=Europe/Rome&forecast_days=1"
```

Parse the `current` object:
- `temperature_2m` → outside_temp_c
- `apparent_temperature` → outside_feels_like_c
- `relative_humidity_2m` → outside_humidity_pct
- `wind_speed_10m` → outside_wind_speed_kmh
- `cloud_cover` → outside_cloud_cover_pct
- `dew_point_2m` → outside_dewpoint_c
- `surface_pressure` → outside_pressure_hpa
- `uv_index` → outside_uv_index
- `shortwave_radiation` → outside_shortwave_radiation_wm2 (W/m², solar radiation hitting the surface)
- `weather_code` → convert WMO code to `outside_condition` string (0=Clear sky, 1=Mainly clear, 2=Partly cloudy, 3=Overcast, 45/48=Fog, 51-57=Drizzle, 61-67=Rain, 71-77=Snow, 80-86=Showers, 95-99=Thunderstorm)
- `hourly.sunshine_duration[current_hour]` → divide by 3600 → outside_sunshine_index (0-1 ratio). If unavailable, empty string.
