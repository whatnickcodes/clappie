Monitor Tailscale and auto-restart if down.

1. Check if Tailscale background is running:
   ```
   tmux has-session -t background-tailscale 2>/dev/null && echo "running" || echo "stopped"
   ```

2. If stopped, start it:
   ```
   clappie background start tailscale
   ```
   Wait a few seconds for it to connect.

3. Verify it's actually connected:
   ```
   tailscale status --json 2>/dev/null | jq -r '.BackendState'
   ```
   Should say "Running".

4. If you had to restart it, send me a Telegram notification:
   ```
   clappie sidekick spawn "telegram user YOUR_ID: Hey, Tailscale was down - I restarted it. Funnel should be back up."
   ```

5. Log to recall/logs/heartbeat/YYYY-MM-DD.txt (APPEND one line to daily file):
   - Format: [HH:MM] ssh-tunnel-monitor ✓ up
   - Or: [HH:MM] ssh-tunnel-monitor ⚡ was down, restarted

   NEVER create standalone files. ALWAYS append ONE line to the daily file.

Don't spam - if you already restarted it this hour, don't send another notification.
